新生物品清单
======

必须品
-------
* 拖鞋  ``学校根本没得买``
* WiFi路由 ``这里最便宜的超过一千SAR``
* 衣服 ``足够多的衣服，这里的衣服都很丑``
* 转接头 ``有的宿舍欧标，有的宿舍英标，混乱``
* 全币种的Visa或者master信用卡 ``刚来没有办银行卡时候使用，不办找人借现金也行。注意是全币种，否则在沙特刷不了``


禁止物品
---------
* 无人机 ``会被沙特海关没收``
* 含酒精的任何物品 ``包括酒心巧克力``
