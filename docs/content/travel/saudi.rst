KAUST 景点
======

接下来讲的大多数是在沙特阿拉伯内部的值得旅游的景点
俗话说“读万卷书，行万里路”。
每一次在沙特旅游，都能让KAUSTian对世界有着不同的见解
也更能理解什么是纯粹的幸福和快乐
