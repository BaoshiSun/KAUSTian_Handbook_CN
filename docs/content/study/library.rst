图书
^^^^^^^^
实体书资源较少，且仅有一排书架，大多为电子资源，且校内网可访问绝大部分科研论文数据库。
图书馆有大量Mac 台式机以供公用，但是仅旧版Mac有windows 和 Mac OS双系统
图书馆内可以免费打印，只需要输入信息门户账号密码即可
