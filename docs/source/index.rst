KAUST生存手册
=============================================
<KAUST生存手册>诞生于2021年1月。本项目旨在将它制作成WIKI文档在线发布，并长期维护，希望能给KAUST在读以及入学新生带来帮助。 本文档将涵盖学习中生活的诸多生存技巧和来自不同作者们的体验。感谢参与本文档编辑的作者们！
